# Mapa de Torres Telefonicas

## Descripcion
Script Python que procesa archivos Excel con coordenadas de torres telefonicas y genera un mapa interactivo HTML con Folium.

## Caracteristicas
- Lee archivos Excel con columnas 'Latitud' y 'Longitud' (case-insensitive)
- Limpia coordenadas con formato de separador de miles incorrecto
- Genera mapa interactivo con iconos de torres
- Muestra circulos de radio configurable divididos en 3 sectores de 120 grados
- Indica puntos cardinales (N, S, E, O) alrededor de cada torre
- Brujula interactiva en esquina superior derecha para orientacion
- Selector de capas: Mapa Estandar, Satelital

## Uso
```bash
python mapa_torres.py ruta/archivo.xlsx
python mapa_torres.py ruta/archivo.xlsx --radio 300
python mapa_torres.py ruta/archivo.xlsx -r 1000
```

## Parametros
- `archivo_excel`: Ruta al archivo Excel (obligatorio)
- `-r, --radio`: Radio del circulo en metros (opcional, default: 500)

## Requisitos del Excel
- Debe contener una hoja llamada "FTD"
- Debe tener columnas con nombres que contengan "Latitud" y "Longitud"

## Dependencias
- pandas
- folium
- openpyxl

## Estructura del Proyecto
- `mapa_torres.py` - Script principal
- `test_torres.xlsx` - Archivo de prueba

## Salida
Genera un archivo HTML con formato `mapa_YYYYMMDD_HHMMSS.html`
